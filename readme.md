# TESTING WITH WEBPACK && Manage project with webpack :boom:


bundle js to 'bundle.js'    
extract css to 'style.css'    
   
   
   
### Install Dependencies
```sh
$ npm install
```
   
### Local development
```sh
$ npm run dev
```
   
### Build production
```sh
$ npm run build
```
   
### Main Dependencies
* webpack
* babel
* sass
* postcss(autoprefix)
