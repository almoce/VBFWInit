# Manage or test project with webpack :boom:
#### webpack simple project boilerplate	
   
### Install Dependencies
```sh
$ npm install
```
   
### Local development
```sh
$ npm run dev
```
   
### Build production    
   
build js and css file in dist dirctory
   
```sh
$ npm run build
```
   
### Main Dependencies
* webpack
* babel
* sass
* postcss(autoprefix)
